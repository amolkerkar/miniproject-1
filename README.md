# Run Project
* In cmd/terminal at project implementation location
	* To Run Project
		1. Run "make"
		2. Run "make run"
	* To Test Unit Testing(Determining the chances at every situation)
		1. Run "make test"
	* To Check the code coverage
		1. Run "make coverage"
	* To delete all the executable files
		1. Run "make clean"
